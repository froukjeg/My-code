 Content 
 -------

 Readme.txt        			: This file.

 ReleaseNotes.txt  			: Release Notes of the last software release.
 
 CAENComm.dll              	: The CAEN C/C++ Comm Library
 
 lib                        : Directory containing the library binary file
                              and an install script .
 include                    : Directory containing the relevant header files.
 
 java						: Directory containing the java library file
                              and a simple java demo.
 
 labview					: Directory containing the Labview Vis library 
                              and a simple Labview demo.
 System Requirements
 -------------------
 - Windows XP/Vista/7
 - CAENVMELib library version 2.40 or above.
 - Labview 2009 or above.


 Installation notes
 ------------------
 
 - Run the CAENCommSetup.exe setup program.
